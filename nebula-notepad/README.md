# WTE Nebula Notepad

A cosmic-themed offline-first notepad built as a single installable HTML file.

## Features
- Draggable/resizable note cards with tabs
- 13 themes including Glossy White, Matte Black, Gold, Chrome, Purplesque
- Optional background image (default galaxy photo) with custom image upload
- Textured glass frames toggle
- Password-locked last tab on each note — safe for keys/private info
- Export/import notes as JSON
- Keyboard shortcuts: Ctrl+N (new note), Ctrl+T (new tab), Ctrl+D (duplicate), Ctrl+C/X/V (copy/cut/paste note), Escape (close)
- PWA support — install to desktop/mobile

## Password-locked tabs
The last tab on any note can be locked with a password. Set the password by clicking the lock icon. The tab body is hidden until unlocked. Useful for hiding API keys, passwords, or private notes.

## Custom backgrounds
Open Settings ⚙ → Background Image. Toggle the default image on/off, upload your own image, or reset to the galaxy default. You can also add theme-specific background images by placing files named `{theme-id}.jpg` or `{theme-id}.png` next to `index.html`.
